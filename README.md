## 1. `npm install`

`npm install`を行います。`npm`のバージョンは Volta で固定していますが、Volta を使っていない人は、`npm -v`で 10 系かどうかを確かめます。

```shell
#npmのバージョンを確認（8系でない場合は、8系をインストール）
npm -v

#パッケージをインストール
npm install
```


## 2. `npm run dev`

`npm run dev`でローカルサーバーを立ち上げます。

```shell
npm run dev
```
